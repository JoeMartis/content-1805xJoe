Clickers

  • We will use the Turning Technologies clickers in class. If you already have one from physics or chemistry it will work here as well.
  • If you need one you can get it at the MIT coop.
  • To register your clicker go to register clicker. Simply follow the directions.
  • If you change clickers you can register the new one. We will download the latest registration information just before class each day.

Matlab

We will use Matlab in this class, so you will need to have it installed on your computer. This is available for free to all students from: Student Matlab. You should download and install it now.

Don't worry if you are not familiar with Matlab, we will provide plenty of tutorials and guidance in its use.

If you are so inclined you can use Octave which is the open source copy of Matlab.
