\prob{5}{4}
\myquest{Suppose we have 8 teams labeled $T_1$, \ldots, $T_8$. 
Suppose they are
ordered by placing their names in a hat and drawing the names out one at a time.
}

a)\myquest{ 
How many ways can it happen that all the odd numbered teams are in 
the odd numbered slots and all the even numbered teams are in the
even numbered slots?
}
\mysol{
Slots 1, 3, 5, 7 are filled by $T_1$, $T_3$, $T_5$, $T_7$ in any order: 4! ways.\nl5
Slots 2, 4, 6, 8 are filled by $T_2$, $T_4$, $T_6$, $T_8$ in any order: 4! ways.\nl5
\ans $4!\cdot 4! = 576.$
}

b)\myquest{ 
What is the probability of this happening?
}
\mysol{
There are 8! ways to fill the 8 slots in any way.\nl5 
Since each outcome is equally likely the probabilitiy is\nl5
$\ds{\frac{4!\cdot 4!}{8!} = \frac{576}{40320} =
0.143 = 1.43\%.}$
}

\prob{7}{3} Textbook page 37, 3.4
\mysol{\nl5
We compute all the pieces needed to apply Bayes' rule.\nl5
We're given $P(T|B) = .7 \myimply P(T^c|B) = .3$, 
\quad $P(T|B^c) = .1 \myimply P(T^c|B^c) = .9$.\nl5
$P(B) = 1.3\times 10^{-5} \myimply P(B^c) = 1 - P(B) = 1 - 1.3\times 10^{-5}$.

We use the law of total probability to compute $P(T)$:\nl5
$P(T) = P(T|B)\,P(B) + P(T|B^c)\,P(B^c) = .1000078.$

Now we can use Bayes' rule to answer the question:\nl5
$\ds{ P(B|T) = \frac{P(T|B)\,P(B)}{P(T)} = 9.10\times 10^{-5}}$, 
\quad $\ds{ P(B|T^c) = \frac{P(T^c|B)\,P(B)}{P(T^c)} = 4.33\times 10^{-6}}$,

\prob{9}{3} Textbook page 37, 3.10
\mysol{\nl5
For a given problem
let $C$ be the event the student gets the problem correct and 
$K$ the event the student knows the answer.

The question asks for $P(K|C)$.

We'll compute this using Bayes' rule:
$\ds{ P(K|C) = \frac{P(C|K)\,P(K)}{P(C)}}.$\nl5
We're given: \, $P(C|K) = 1$, \, $P(K) = 0.6$.\nl5
Law of total prob.: \nl5 
$P(C) = P(C|K)\,P(K) \, + \, P(C|K^c)\,P(K^c)
= 1\cdot 0.6 \, + \, 0.25\cdot 0.4 = 0.7$.\nl5
Therefore $\ds{P(K|C) = \frac{0.6}{0.7}  = .857 = 85.7\%}$.
}
